# Hundir-La-Flota
Proyecto en el que se crea desde cero el famoso juego Hundir La Flota.

## Simbología del tablero

| Símbolo | Significado |
|---------|-------------|
| `_`     | Celda sin disparar |
| `O`     | Barco propio |
| `X`     | Tocado |
| `·`     | Agua (disparo fallido) |
| `S`     | Hundido |
